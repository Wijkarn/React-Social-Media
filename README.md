# Falcon: My React social media app.

## To start

Run these commands in terminal
```bash
cd server
npm run dev
```

Start a new terminal in project
```bash
cd client
npm start
```

Fill database
```bash
cd server
npm run populate
```

## To stop double render

Disable (comment out) React Strict Mode in index.js
